How to Install (Android)

  1. Open Google Play (recommended) or download the official APK from the official site.
  2. If APK: enable Install unknown apps for your browser (Settings → Apps → Special access).
  3. Run the APK installer and follow on-screen steps.
  4. Launch the game, let additional resources download, then sign in (Google/Facebook/Guest).
Security Warning: If you see an unknown source warning, double-check the download URL and checksum before proceeding.

Frequently Asked Questions

Is it safe to install from third-party sites?

Prefer official sources (Play Store or official website). If using an APK, verify checksum and use reputable mirrors.

Why is the game large after install?

The initial installer is often a launcher; additional game files (maps, voice packs) download after first launch.

Which account should I use?

Use Google or a linked social account for persistent progress; Guest accounts can be lost if you clear app data.
